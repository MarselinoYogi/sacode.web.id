# SaCode.web.id

Our website is under development progress.

<b>Setup Project</b>

`composer install`

`npm install`

`cp .env.example .env`

create database and setup on the `.env` file

`php artisan key:generate`

`php artisan migrate`

`php artisan serve`

`npm run dev`
